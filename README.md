E-Commerce Sales Analysis Dashboard
A professional, interactive Excel dashboard created to visualize and analyze business performance across major cities in India. This project demonstrates skills in data cleaning, pivot tables, and interactive data visualization.
✨ Key Features:
Interactive Slicers: Allows users to filter data instantly by City (Bangalore, Chennai, Delhi, Hyderabad, Mumbai) and Product.
Dynamic Bar Charts: A dark-themed horizontal bar chart for clear visualization of Sales vs. Profit without clutter.
Tabular Reports: Formatted Pivot Tables providing summarized insights into regional performance.
Clean UI/UX: A professional interface with a plain white background (no gridlines) and a dedicated header.
💡 Business Insights:
Hero Product: Laptops are the top-performing category, generating the highest sales revenue across all cities.
Region Performance: Delhi and Hyderabad show very consistent profit margins, making them key markets.
Interactivity: The dashboard enables real-time deep-dives into specific product trends with a single click.
🛠️ Tools Used:
Microsoft Excel: Pivot Tables, Pivot Charts, Slicers, and Conditional Formatting.
Data Storytelling: Transforming raw sales rows into actionable business insights.
